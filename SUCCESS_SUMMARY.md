### 🔍 根本问题
32位掩码操作在IAR EW8051中有问题：
```cpp
content &= ((1U << len) - 1);  // 当len=32时有问题
```

### 🔧 解决方案
修复32位掩码操作：
```cpp
if (len == 32) {
  // 32位数据不需要掩码
} else {
  content &= ((1U << len) - 1);
}
```
